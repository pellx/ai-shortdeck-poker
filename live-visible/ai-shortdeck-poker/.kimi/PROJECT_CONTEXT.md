# ai-shortdeck-poker 项目上下文

## 项目概述

这是一个 **AI 德州扑克直播互动面板** 的 React + Vite 项目。左侧 380px 固定区域包含**弹幕显示**和**猜对排行榜**，右侧区域由另一个 agent 负责开发牌桌对局界面。

项目部署为 OBS 浏览器源 / 直播叠加层使用，`index.html` 背景透明。

## 技术栈

- React 19 + Vite 8
- bilive-message-listener / tiny-bilibili-ws（已安装但未接入真实 API，当前使用模拟数据）
- 开发服务器：`npm run dev`（默认 5173，占用时自动递增）

## 文件结构

```
src/
  main.jsx       ← 应用入口：左侧弹幕+排行榜面板 + 右侧牌桌区域
  danmuka.jsx    ← 弹幕组件：模拟弹幕从上向下滚动
  RankList.jsx   ← 排行榜组件：领奖台(前三名) + 展开列表(4-10名)
  App.jsx        ← 右侧牌桌占位（当前只显示"111"），由另一个 agent 负责
  App-example.jsx, App.css, index.css  ← 未使用
public/
  favicon.svg, icons.svg
dist/            ← 构建输出
vite.config.js   ← 含 B站 API 代理配置
```

## 已实现功能

### 1. 弹幕系统（danmuka.jsx）

- **滚动方向**：从上向下，新弹幕插入到数组开头（最顶部），旧弹幕向下推
- **模拟数据**：15个预设用户 + 25条预设弹幕内容，每 800ms 自动生成一条
- **动画**：新弹幕入场 `danmuSlideDown 0.1s ease-out`（`translateY(-10px)` → `translateY(0)`）
- **消息上限**：最多保留 30 条，超出时从末尾删除最旧消息
- **主题支持**：弹幕气泡卡片背景/边框/颜色随主题变化

### 2. 排行榜系统（RankList.jsx）

- **数据**：10 条测试数据（用户 + 猜对次数），按次数降序排列
- **领奖台**：前三名以体育赛事领奖台形式展示（🥇金色/🥈银色/🥉铜色底座）
- **展开/收起**：第 4-10 名列表通过 `max-height` + `opacity` 平滑过渡显示/隐藏
- **无标题栏**：仅显示排名内容

### 3. 统一背景容器（main.jsx）

- 弹幕 + 排行榜共享一个圆角 16px、毛玻璃背景的大容器
- 背景色/边框随 `?theme=` 参数变化

### 4. 底部渐变遮罩（danmuka.jsx）

- **收起时**：遮罩高度 `0px`，弹幕完全可见
- **展开时**：遮罩高度 `30px`，底部弹幕从透明渐变到面板背景色淡出
- **分隔线**：遮罩 div 底部自带 `borderBottom` 分隔线，增强弹幕区与排行榜的区分度
- **动画**：`transition: height 0.4s ease`

### 5. 自动切换机制（main.jsx）

- 每 **5 秒** 自动在「收起」↔「展开」间切换
- `isExpanded` 状态在 `LeftPanel` 中统一管理，通过 props 同步传给 Danmu 和 RankList

## URL 参数

| 参数 | 说明 | 默认值 |
|---|---|---|
| `?theme=dark` | 主题：dark / light / blue / purple / green | dark |
| `?roomId=xxx` | 房间号（当前仅用于调试显示） | 模拟房间 |

## 主题系统

`PANEL_THEMES`（main.jsx）定义了 5 种主题的面板背景、边框、分隔线颜色：

| 主题 | 背景 | 分隔线 |
|---|---|---|
| dark | `rgba(18, 18, 26, 0.92)` | `rgba(255,255,255,0.08)` |
| light | `rgba(250, 248, 245, 0.92)` | `rgba(0,0,0,0.08)` |
| blue | `rgba(10, 25, 50, 0.92)` | `rgba(100,180,255,0.2)` |
| purple | `rgba(35, 15, 45, 0.92)` | `rgba(200,120,255,0.18)` |
| green | `rgba(20, 64, 27, 0.85)` | `rgba(78,217,101,0.3)` |

弹幕和排行榜各自也有独立的 `THEMES` 定义文字颜色、卡片背景等。

## 协作关系

- **本 agent 负责**：左侧 380px 弹幕 + 排行榜互动面板
- **另一个 agent 负责**：右侧 `App.jsx` 牌桌对局界面（当前为占位）
- 两者通过 `main.jsx` 中的左右 flex 布局共存，互不干扰

## 待接入的真实数据（未来工作）

- B站真实弹幕：当前使用 `blive-message-listener` / `tiny-bilibili-ws` 已安装但未接入
- 真实猜对次数：当前使用 `TEST_RANK_DATA` 静态数据
- 对局状态控制：当前用 `setInterval` 模拟 5s 切换，未来应由牌桌 agent 通过 props 或全局状态传递真实对局阶段

3. `index.html` 设置了 `background: transparent`，适合作为 OBS 浏览器源
